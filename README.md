# OffsetPotential-abrupt-vs-gradual
Fernández-Folgueiras, U., Veiga-Zarza, E., Kessel, D., Cipriani, G. A., Álvarez, F., Alba, G., &amp; Carretié, L. (in preparation). Impact of the abrupt offset of emotional stimulation on brain activity.

## Repository structure

This repository contains the stimulation and analysis scripts for two EEG experiments examining the neural processing of emotional stimulus disappearance.

## Data

Individual averaged ERP and behavioral data are available at: https://osf.io/d26nk/

## Stimuli

Emotional images belong to the EmoSelec subset of the EmoMadrid database (Carretié et al., 2019): https://www.psicologiauam.es/CEACO/EmoMadrid.htm

## Requirements
- MATLAB R2020b or later
- Psychtoolbox 3 (stimulation scripts)
- FieldTrip and EEGlab (analysis scripts)
